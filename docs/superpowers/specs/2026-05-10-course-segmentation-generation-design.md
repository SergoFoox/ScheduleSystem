# Специфікація дизайну: Генерація розкладу за курсами (Course Segmentation)

**Дата:** 2026-05-10
**Статус:** Схвалено (реалізовано в коді)
**Технологічний стек:** Timefold Solver 1.33.0, Vaadin Tabs, React.

## 1. Огляд
Для великих навчальних закладів генерація розкладу для всіх груп одночасно може бути ресурсозатратною. Модуль сегментації дозволяє ділити процес на етапи за курсами навчання.

## 2. Користувацький інтерфейс

### 2.1. Вкладки курсів (Course Division Tabs)
*   У діалоговому вікні генерації додано вкладки (наприклад, "1 курс", "2 курс", "Всі").
*   При виборі конкретного курсу список груп для генерації автоматично фільтрується.

## 3. Логіка генерації

### 3.1. Параметризація солвера
*   Ендпоінт `ScheduleService.solve()` приймає список ідентифікаторів груп.
*   Timefold Solver завантажує в робочу пам'ять (Solution) тільки ті заняття та обмеження, що стосуються вибраних груп.

### 3.2. Переваги підходу
*   **Продуктивність**: Значне прискорення пошуку оптимального рішення.
*   **Локальні зміни**: Можливість перегенерувати розклад тільки для першокурсників, не змінюючи вже погоджений розклад старших курсів.
