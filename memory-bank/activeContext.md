# Активний контекст — ASMS V3

## Поточний фокус
Зараз проект знаходиться на стадії завершення основних довідників та переходу до налаштування навчального процесу (Предмети та Матриця компетенцій).

## Останні зміни
1.  **Повний редизайн під PDF-стандарт**: Головний дашборд тепер відображає розклад у вигляді академічної матриці (Групи в стовпцях, Дні/Пари в рядках), що повністю відповідає реальному розкладу коледжу.
2.  **Впровадження Tailwind CSS**: Додано підтримку Tailwind для гнучкої стилізації, оскільки стандартних класів Aura Theme було недостатньо для складної сітки.
3.  **Стабілізація CRUD**:
    *   Реалізовано повноцінне керування Викладачами (включно з лімітами днів та спеціалізацією).
    *   Реалізовано керування Групами з можливістю призначення куратора.
    *   Реалізовано керування Аудиторіями з урахуванням типів приміщень.
    *   Додано повноцінний розділ «Дисципліни» (Предмети).
4.  **Матриця компетенцій**: Впроваджено інтерфейс для зв’язку викладачів із предметами з урахуванням типів занять та пріоритетів.
5.  **Навчальні плани**: Створено окремий розділ для масового перегляду та редагування навантаження всіх груп.
6.  **Інтеграція Timefold**: Кнопка «Згенерувати» тепер запускає реальний процес оптимізації на бекенді, результати якого автоматично відображаються в новій сітці.

## Стан системи
- **Бекенд**: Стабільний. Всі ендпоінти працюють у режимі `@AnonymousAllowed` для полегшення розробки.
- **Фронтенд**: Повністю реактивний (Signals). Білд проходить успішно.
- **База даних**: H2 (файлова), структура актуальна.

## Наступні пріоритети
1.  Впровадження експорту розкладу у PDF/HTML.
2.  Додавання системи автентифікації.
